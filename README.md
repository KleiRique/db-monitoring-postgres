# Database Monitoring & Performance Analysis (PostgreSQL)

Projeto de estudo focado em práticas de DBA utilizando SQL e Python.

## Objetivo
Demonstrar conhecimento em:
- Modelagem de banco de dados
- SQL para análise e diagnóstico
- Organização de projeto voltado a DBA

## Tecnologias
- PostgreSQL
- SQL
- Python

## Estrutura do projeto
- `database/` → scripts de criação do banco
- `queries/` → queries de análise
- `scripts/` → automações em Python
